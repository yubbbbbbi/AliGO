# ALiGO (알리고)

사용자가 특정 매장(전국 다이소, 올리브영 매장) 근처에 도착하면 저장해둔 메모를 자동으로 푸시 알림으로 띄워주는 위치 기반 스마트 메모 서비스입니다.

## 프로젝트 소개
* **문제 정의**: 일상에서 필요한 물건을 메모해도 정작 매장에 방문했을 때 기억하지 못하는 불편함 해소
* **솔루션**: Geofencing API를 활용하여 매장 반경 진입 시점에 메모 알림 제공

---

## 주요 구현 기술

### 1. 지오펜싱(Geofencing) 및 백그라운드 최적화
* **ENTER 이벤트 감지**: 사용자가 매장 반경 진입 시 정밀하게 이벤트를 트리거하여 실시간 알림 전송
* **Foreground Service**: 안드로이드 시스템에 의해 프로세스가 종료되지 않도록 유지하여 안정적인 백그라운드 감지 보장
* **배터리 최적화**: 지속적인 GPS 추적 대신 이벤트 기반 동작으로 전력 소모 최소화

### 2. 매장 후보군 선별 알고리즘
* **탐색 성능 고도화**: 전국 매장 DB 전체를 등록하는 대신, 사용자 위치 기반의 후보군(상위 30개)만 추려 Geofence 등록
* **계층적 필터링**: 지역 단위 필터링 및 분할 탐색을 통해 연산 비용 절감
